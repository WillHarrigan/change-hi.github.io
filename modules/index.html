4. High Performance Computing

This workshop focuses on utilization of High Performance Computing (HPC) clusters, such as Mana, for deep learning tasks.

It also covers the benefits of understanding the different types of file systems available on HPC clusters and basic ways one would stage data from slower file systems to faster file systems.

5. Python and Jupyter

This module provides recommended readings and introductory tutorials on Python and Jupyter.

Python and Jupyter are prerequisites for many of the workshops in CI-TRACS. This module will help bring you up to speed with these technologies if you are not already familiar with them.

13. Smart Data Collection for Sensor Networks

This workshop presents the Tapis framework and how to use its APIs to store and process streaming sensor data.

It also covers the tapipy Python library to aide in interacting with the Tapis API services within an interactive Jupyter notebook. The concepts of containers and Actors will also be introduced at a high level.
